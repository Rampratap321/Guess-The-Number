🎯 Guess The Number

A simple and fun number guessing game built using HTML, CSS, and JavaScript.

The computer randomly generates a number, and the player has to guess it correctly. After each guess, the game provides feedback indicating whether the guessed number is too high or too low. The goal is to find the correct number in as few attempts as possible.

🚀 Features
Random number generation
Interactive user interface
Instant feedback for each guess
Tracks the number of attempts
Simple and responsive design
Restart/New Game functionality
🛠️ Technologies Used
HTML5 – Structure of the application
CSS3 – Styling and layout
JavaScript (ES6) – Game logic and interactivity

🎮 How to Play
Enter a number in the input field.
Click the Guess button.
The game will tell you whether your guess is:
Too High 📈
Too Low 📉
Correct 🎉
Keep guessing until you find the correct number.
Start a new game and try to beat your previous score.
